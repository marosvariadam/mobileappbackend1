# Mobile App Backend

Ez a repository a szakdolgozatom szerveroldali komponensét tartalmazza. Az alkalmazás egy cross-platform mobilapplikációt szolgál ki, biztosítva az adatkezelést és az üzleti logika végrehajtását.

## Technológiai háttér

A projekt a következő technológiák felhasználásával készült:

- Nyelv: C#
- Keretrendszer: ASP.NET Core Web API
- Adatbázis: MongoDB (NoSQL)
- Architektúra: RESTful API, rétegelt architektúra (Controller-Service-Repository)

## Funkcionalitás

Az API az alábbi fő funkciókat látja el:

- Felhasználókezelés és autentikáció
- Adatbázis műveletek (CRUD) a MongoDB gyűjteményeken
- Hibakezelés és bemeneti adatok validálása
- JSON formátumú adatkommunikáció a klienssel

# Mobile App Backend

This repository contains the server-side component of my thesis project. The application serves a cross-platform mobile application, handling data management and business logic execution.

## Technical Stack

The project is built using the following technologies:

- Language: C#
- Framework: ASP.NET Core Web API
- Database: MongoDB (NoSQL)
- Architecture: RESTful API, layered architecture (Controller-Service-Repository)

## Functionality

The API provides the following main functions:

- User management and authentication
- Database operations (CRUD) on MongoDB collections
- Error handling and input data validation
- JSON data communication with the client
